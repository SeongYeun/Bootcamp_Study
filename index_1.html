## 1. 기본 선택자 1) 태크 선택자 document.querySelectorAll("li") 2) Class 선택자 document.querySelectorAll(".orange") 3) ID 선택자 document.querySelectorAll("#orange") ## 2. 복합 선택자 (태그명 + class명 조합) 1) 일치 선택자 document.querySelectorAll("li.orange") 2) 후손 선택자 document.querySelectorAll("li .orange") 3) 자식 선택자 document.querySelectorAll("li >.orange") 4) 인접형제 선택자 document.querySelectorAll(".orange + li") 5) 일반형제 선택자 document.querySelectorAll(".orange ~ li") ## 3. 가상클래스 선택자 # 3-1. 사용자 행동에 따른 변화 1) hover document.querySelectorAll("a:hover") 2) active document.querySelectorAll("a:active") 3) focus document.querySelectorAll("a:focus") # 3-2. 요소 상황 (.class명 태그명:요소) 1) first-child document.querySelectorAll(".fruits span:first-child") ㅡ 태그는 해당 class의 첫번째 자식의 태그명을 써야 함 2) last-child document.querySelectorAll(".fruits h3:last-child") ㅡ 태그는 해당 class의 마지막 자식의 태그명을 써야 함 3) nth-child document.querySelectorAll(".fruits *:nth-child(3)") ㅡ 해당 class의 3번째 자식 태그의 값 추출 . document.querySelectorAll(".fruits *:nth-child(2n)") ㅡ 해당 class의 짝수번째 자식 태그들의 값 추출 (0부터 계산) . document.querySelectorAll(".fruits *:nth-child(2n+1)") ㅡ 해당 class의 홀수번째 자식 태그들의 값 추출 (0번째 포함) 4) not document.querySelectorAll(".fruits *:not(span)") ㅡ 해당 class에서 span태그를 제외한 나머지 태그들의 값 추출 ## 4. 가상요소 선택자 1) before document.querySelectorAll(".fruits::before{content: "앞!";}") 2) after document.querySelectorAll(".fruits::after{content: "뒤!";}") ## 5. 속성 선택자 1) disabled document.querySelectorAll("[disabled]") 2) readonly document.querySelectorAll("[readonly]") 3) password document.querySelectorAll("[type='password']")
