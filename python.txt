> 입력 : input()

> 객체
1) 가변 : list, dictionary, set
2) 불변 : tuple, 숫자, 문자열  -> 변경시 신규 객체가 생성되는 것임


> 기본 문법 w/ 관용적 문법
상수의 변수명 : 모두 대문자
snake_case : 단어사이 모두 언더바(_)로 연결 >>> 함수, 변수
PascalCase : 단어 첫글자마다 대문자로 작성 >>> 클래스 사용
cAMELcASE  : 단어 첫글자마다 소문자로 작성
